# marketing-site
Landing page and client funnel, lead to proposal

## Setup

Install dependencies and start the dev server:

```bash
npm install
npm run dev
```

The site runs at http://localhost:3000 by default.

This project is a minimal Astro + React scaffold. Files added:
- astro.config.mjs
- package.json
- src/pages/index.astro
- src/pages/about.astro
- src/components/Layout.astro
- src/components/Header.jsx
- public/favicon.svg
